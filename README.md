Amazon Prime Content Analytics Dashboard
Project Overview

This project analyzes Amazon Prime Movies and TV Shows using Tableau to uncover insights related to:

Content growth trends
Genre popularity
IMDb rating distribution
Audience engagement
Regional content patterns
Content performance segmentation

The dashboard was designed with a dark-theme interactive interface focused on storytelling and business insights.

Tools Used
Tableau
Excel / CSV
Data Cleaning
Data Visualization
Dashboard Design
Key Features
KPI cards for total titles, ratings, genres, and countries
Content growth trend analysis
IMDb rating distribution
Engagement vs popularity analysis
Genre performance breakdown
Interactive filters and dashboard actions
Business Insights
Drama and Comedy dominate platform content
Recent years show aggressive content expansion
High IMDb ratings do not always translate to high engagement
Certain genres consistently outperform in audience retention
